## ğŸ“Š CLASSIFICATION LEVELS (4 Types)

### 1. **PUBLIC** (Risk: LOW)
- README files, marketing materials, public docs
- Empty files, whitespace only
- No restrictions, no logging

### 2. **INTERNAL** (Risk: LOW-MEDIUM)
- Internal memos, staff updates
- System logs (non-sensitive)
- Allowed with logging

### 3. **CONFIDENTIAL** (Risk: HIGH)
- Customer data, financial reports
- Meeting minutes, contracts
- Source code, architectural diagrams
- User gets warning popup

### 4. **RESTRICTED** (Risk: CRITICAL)
- API keys, passwords, private keys
- Database dumps with PII
- Trade secrets, board documents
- Blocked by default (admin override possible)
